# SDK Introduction

The Bike Matrix SDK (with Web Components) is available through [NPM](https://www.npmjs.com/package/@bikematrix/web-components/).

To use the Bike Matrix SDK you will need to:

- [Install the Bike Matrix NPM package](docs/sdk-integration/installation.md).
- [Setup configuration for the Bike Matrix SDK](docs/sdk-integration/configuration.md).
- [Setup authentication with the configuration](docs/sdk-integration/authentication.md).
- [Integrate the Bike Matrix NPM package into your solution](docs/sdk-integration/integration.md).
- [Implement the web components into your solution.](docs/sdk-integration/web-components/overview.md)

If you want to add more custom integrations. You could:

- Utilise the Bike Matrix [HTML Events](docs/sdk-integration/html-events.md).
- Utilise the Bike Matrix [API directly](docs/api-integration/introduction.md).
