# Bike Selector Buttons

You must implement one or more buttons to open the [Bike Selector](docs/sdk-integration/web-components/bike-selector.md).

Here are your options:

## Custom Button

You can implement a custom button to open the bike selector.

```html
<button id="bm_selectBikeButton">Select Bike</button>
```

## Bikon

You could use our [Bikon](docs/sdk-integration/web-components/bike-selector-buttons/bikon.md).
