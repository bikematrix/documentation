# Bike Matrix Metafield

The Bike Matrix metafield is an additional field on each product used to store part numbers that are recognised by the Bike Matrix database.

If you use a 12- or 13-digit barcode, or the original manufacturer’s part number, in either the SKU or Barcode field, Bike Matrix will check these during an automatic overnight sync process or when you first add a collection to the Bike Matrix application in Shopify. When a match is found in the Bike Matrix database, the recognised part number is automatically copied into the Bike Matrix metafield.

This approach allows Bike Matrix to collate recognised part numbers into a single field, rather than relying on multiple product fields.

The Bike Matrix metafield can also be updated manually. This is useful if your SKU or Barcode fields are reserved for internal references or distributor-specific part numbers.
