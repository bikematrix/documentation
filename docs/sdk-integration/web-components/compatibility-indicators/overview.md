# Compatibility Indicators

In order to display the compatibility status of products, you should implement our compatibility indicators.

Here are your options:

## Collection Result

The [Collection Result](docs/sdk-integration/web-components/compatibility-indicators/compatible-list.md) provides a compatibility result under each product in a collection.
